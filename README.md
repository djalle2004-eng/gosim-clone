# 🌍 SoufSim eSIM Marketplace - المنصة المتكاملة لشرائح الـ eSIM

منصة **Enterprise-Grade** متكاملة للتجارة الإلكترونية متخصصة في بيع وتزويد شرائح الاتصال الإلكترونية (eSIM)، مبنية بهيكلية موزعة (Event-Driven) تضمن الموثوقية العالية والأداء الفائق.

---

## 🚀 المواصفات التقنية (Advanced Tech Stack)

تعتمد المنصة على بنية **Monorepo** مع فصل كامل للخدمات لضمان القابلية للتوسع (Scalability).

### 💻 الواجهة الأمامية (Frontend)

- **Framework**: React 18 مع Vite.
- **State Management**: **React Query** للبيانات الخارجية و **Zustand** للحالة المحلية والسلة.
- **Visuals**: **Framer Motion** للتحريكات، **Tailwind CSS** للتصميم، و **Lucide React** للأيقونات.
- **Charts & Data**: **Recharts** للتحليلات المتقدمة و **React Simple Maps** للخرائط التفاعلية.
- **Real-time**: **Socket.io-client** لاستقبال الإشعارات الفورية.

### ⚙️ الظهر البرمجي (Backend)

- **Runtime**: Node.js مع TypeScript.
- **Framework**: Express.js مع هندسة معمارية قائمة على الخدمات (Service-Layer Architecture).
- **Automation & Queues**: **BullMQ** (مدعوم بـ Redis) لإدارة المهام الخلفية (الرسائل، التزويد، الويب هوك).
- **Real-time Engine**: **Socket.io** للتواصل اللحظي مع المستخدمين.
- **Scheduled Jobs**: **node-cron** للمزامنة الدورية (أسعار الصرف، حالة الشرائح، التنبيهات).
- **ORM**: Prisma مع PostgreSQL.

## ✨ الخصائص والميزات الاحترافية

### 🔔 نظام الإشعارات والأتمتة (Event-Driven System)

- **قنوات متعددة**: إشعارات داخل التطبيق (In-app)، رسائل بريد إلكتروني احترافية (React Email)، وتنبيهات المتصفح (Web Push - VAPID).
- **ذكاء اصطناعي في الأتمتة**:
  - تنبيهات تلقائية عند وصول استهلاك البيانات لـ 80%.
  - رسائل تذكير قبل انتهاء صلاحية الشريحة بـ 3 أيام.
  - مزامنة تلقائية لأسعار الصرف كل 6 ساعات لضمان دقة الأسعار.

### 📈 النظام المالي والمحاسبي (Finance & Accounting System)

- **لوحة القيادة المالية (Financial Hub)**: مراقبة لحظية للـ **MRR**، **ARR**، **الربح الصافي**، و **Gross Margin**.
- **نظام عمولات الموزعين (Reseller Commissions)**:
  - حساب آلي للعمولات عند كل طلب.
  - إدارة المدفوعات (Payouts) وتصدير التقارير الشهرية للشركاء.
- **إدارة الضرائب والفواتير (Tax & Invoicing)**:
  - توليد تلقائي لفواتير **PDF** احترافية تحتوي على **QR Code** للتحقق.
  - دعم ضريبة القيمة المضافة (**VAT**) حسب كل دولة مع نظام الـ Reverse Charge للشركات.
  - تقارير ضريبية مجمعة وقابلة للتصدير (Excel).

### 🛡️ مركز القيادة للإدارة (Premium Admin Panel)

- **لوحة تحكم ذكية**: عرض مؤشرات الأداء (KPIs) مثل MRR، AOV، ونسبة نجاح المدفوعات مع رسوم بيانية لحظية.
- **مركز التحليلات (Analytics Hub)**:
  - **Cohort Analysis**: لفهم معدل احتفاظ المستخدمين (Retention).
  - **LTV (Lifetime Value)**: تحليل القيمة الدائمة لكل عميل.
  - **Conversion Funnel**: لتتبع رحلة العميل من الزيارة إلى الشراء المتكرر.
  - **Geographic Heatmap**: عرض كثافة المبيعات عالمياً على الخريطة.
- **إدارة العمليات**: نظام استرجاع الأموال (Stripe Refund) بضغطة زر، وإدارة متقدمة للمستخدمين والطلبات.

### 🤝 بوابة الشركاء (Partner & Developer Portal)

- **API Management**: إنشاء وإدارة مفاتيح API مع تحديد الصلاحيات (Scopes).
- **Webhooks**: إعداد روابط التنبيهات مع توقيع أمني (HMAC-SHA256) لضمان موثوقية البيانات.
- **API Playground**: بيئة تجريبية للمطورين لاختبار التكامل مباشرة من المتصفح.

### 👤 تجربة المستخدم (User Experience)

- **محفظة إلكترونية (Wallet)**: دعم الرصيد الداخلي والمكافآت.
- **نظام الإحالة (Referral System)**: كسب رصيد عبر دعوة الأصدقاء.
- **فواتير المستخدم**: إمكانية تحميل الفواتير الرسمية مباشرة من سجل الطلبات.
- **Checkout سلس**: دعم Stripe للدفع العالمي وCIB للدفع المحلي مع واجهة دفع مؤمنة بالكامل.

## 🛠️ التشغيل والتطوير (Setup & Dev)

### المتطلبات

- Node.js >= 20
- Redis Server (للجداول والمهام)
- PostgreSQL
- SMTP Credentials (للبريد)

### خطوات التشغيل السريع

1. **تثبيت المكتبات**: `npm install`
2. **إعداد البيئة**: قم بنسخ `.env.example` إلى `.env` في كل من الـ `frontend` والـ `backend`.
3. **البنية التحتية**: `docker-compose up -d`
4. **تهجير البيانات**: `npx prisma migrate dev`
5. **التشغيل**: `npm run dev`
